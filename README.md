# specification

统一项目代码规范的命令行工具

## 安装

```bash
npm i -g @game/specification
```

## 使用方法

```bash
spec -p /xxx/xxx
```

## 选项

|   名称    |   类型   | 必填 |        默认值        |                                     描述                                     |
| :-------: | :------: | :--: | :------------------: | :--------------------------------------------------------------------------: |
| -p/--path | `String` | `N`  | `运行命令的当前目录` | 要统一规范的工作目录，支持相对路径和绝对路径，相对路径相对当前运行命令的目录 |
